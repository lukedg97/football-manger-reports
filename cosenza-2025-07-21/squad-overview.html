How the scores work

Each composite score is a league-wide percentile rank (0-100) — a score of 70 means the player outperforms 70% of outfield players across all 26 clubs. Scores require 270+ minutes. Colour: 50-69 = above average, 70+ = top tier, red = below average.

Attacking Value = Attacking Value: 40% Non-Penalty xG/90, 35% xA/90, 15% Goals/90, 10% Assists/90
Creation Value = Creation Value: 35% Key Passes/90, 30% Progressive Passes/90, 20% xA/90, 15% Dribbles/90
Defensive Value = Defensive Value: 25% Tackles Won/90, 25% Interceptions/90, 20% Aerial Duels Won/90, 10% Key Tackles/90, 10% Pressure Success/90, 5% Possession Won/90, 5% Clearances/90
Overall Value = Overall Value: position-weighted blend of Attacking Value + Creation Value + Defensive Value. Weights differ by role so that a CB is not penalised for low Attacking Value and a striker is not penalised for low Defensive Value:
